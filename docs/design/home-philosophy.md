# ホーム Philosophy セクション

## 背景表現

- Philosophy セクションの全面に、10px 四方・5px 間隔の角丸タイルを等間隔で配置する。
- タイル色は適用中テーマの `--accent-color` を使用し、テーマ変更時に即時追従する。
- 点滅していない通常タイルは、不透明度 0.1〜0.15% のほぼ見えない状態にする。ランダムに選ばれたタイルは約 1.3〜2.6 秒でフェードイン／フェードアウトさせる。
- fine pointer のカーソル位置を芯とし、72px を基準寸法とする範囲を点灯する。真円にはせず、下側を短く上側を細長くした炎状の輪郭にする。単一タイルだけではなく範囲内の複数タイルを描画し、中心を最も強く、外側ほど滑らかに透明へ近づける。タイル間の隙間にカーソルがあっても周辺点灯を維持する。
- 炎状の輪郭には複数周期の角度波、タイル座標に連続する空間波、上方ほど強くなる左右の揺れを重ねる。輪郭と内部の熱流を毎フレーム連続更新し、完全な球形や一定周期の膨張収縮に見えない自然な揺らぎを作る。`prefers-reduced-motion` が有効な場合は時間変化を止め、非対称な静止形状だけを維持する。
- カーソル周囲の各タイルはそれぞれ独立した点灯強度を持つ。各タイルの基準値を 82% とし、約 64〜100% の範囲を個別の約 0.5〜1.2 秒周期でランダムに往復する。値を瞬時に切り替えず、`smootherstep` のフェードで連続的に変化させる。
- カーソル移動で点灯範囲から外れたタイルは即時に消去せず、個別の明るさを保ちながら約 300〜650ms で透明へ減衰させる。新しい範囲のタイルも約 260〜500ms かけてフェードインし、移動前後の点灯範囲を滑らかにつなぐ。
- カーソルがセクションまたはブラウザ表示領域から外れた場合とウィンドウがフォーカスを失った場合は、約 320ms で点灯範囲をフェードアウトする。`pointerleave` に加えて捕捉段階の `pointerout` と描画中の `:hover` 状態を監視し、ブラウザ枠やスクロールバー側へ抜けて終了イベントを取りこぼしても最後の位置へ点灯を残さない。完了後はカーソル座標を破棄し、再入場まで点灯を残さない。

## パフォーマンスとアクセシビリティ

- 背景は DOM 要素を大量生成せず `canvas` で描画する。静止グリッドは別 canvas にキャッシュし、アニメーション中は変化するタイルだけを重ねる。
- Philosophy セクションが画面外にある間、またはページが非表示の間は描画ループを停止する。
- `prefers-reduced-motion: reduce` が有効な場合はランダム点滅を停止し、静止グリッドとカーソルによる明示点灯だけを表示する。
- canvas は装飾として `aria-hidden="true"` にし、コンテンツや操作要素より背面に配置する。
